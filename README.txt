=======
Occiput
=======

Tomographic inference. 
Typical usage looks like this::

    #!/usr/bin/env python

    import occiput
    # ...


Installation
============

Linux, MacOsX
-------------

There are two ways to install under Linux and MacOsX: 

1. pip install occiput

2. python setup.py
